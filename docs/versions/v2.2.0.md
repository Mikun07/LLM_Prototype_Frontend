## Plain-Language Summary

Version `v2.2.0` makes the version switching policy more explicit and improves the helper
script so the project can move between any documented version variation.

This means you can switch between versions like:

| Example | Meaning |
|---|---|
| `v1.1.0` to `v1.2.0` | Future version-1 variation switching, if those tags are created |
| `v2.0.0` to `v2.1.0` | Move from the frozen version-2 baseline to the first adjustment |
| `v2.1.0` to `v2.2.0` | Move from the policy adjustment to the variation-switching update |
| `v2.2.0` to `v2.0.0` | Return to the frozen version-2 baseline |

Only existing Git tags can be selected. Run `npm run version:list` to see what exists.

## What Changed

| Area | Details |
|---|---|
| Version helper | `scripts/use-version.ps1` now resolves shorthand like `v2.2` and `2.2` to `v2.2.0` |
| Error messages | Missing versions now show the available version tags |
| Documentation | `README.md`, `docs/VERSIONING.md`, `docs/COMMANDS.md`, and version docs explain variation switching |
| Current version | `package.json`, lockfile, and visible app version move to `2.2.0` |

## Supported Switch Formats

These are equivalent when the `v2.2.0` tag exists:

| Input | Resolved tag |
|---|---|
| `v2.2.0` | `v2.2.0` |
| `v2.2` | `v2.2.0` |
| `2.2` | `v2.2.0` |

The same pattern applies to future variations such as `v2.3.0`, `v3.1.0`, or `v1.2.0`.

## Common Commands

List versions:

```powershell
npm run version:list
```

Switch to the current version:

```powershell
npm run version:use -- -Version v2.2 -CleanIgnored -Install
```

Switch to the previous v2 variation:

```powershell
npm run version:use -- -Version v2.1 -CleanIgnored -Install
```

Switch to the frozen v2 baseline:

```powershell
npm run version:use -- -Version v2.0 -CleanIgnored -Install
```

Switch to the frozen v1 baseline:

```powershell
npm run version:use -- -Version v1 -CleanIgnored -Install
```

Create an editable branch from a variation:

```powershell
npm run version:use -- -Version v2.2 -Branch work/from-v2.2.0 -Install
```
