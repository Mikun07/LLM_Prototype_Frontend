## Summary

`v1.0.0` is the first clean frontend baseline for the ReqSmell prototype. It establishes
the React, TypeScript, Vite, Redux Toolkit, Tailwind CSS, Recharts, jsPDF, and Vitest
environment described in `.AI_tool/frontend_README.md`.

This version is not the full ReqSmell product implementation yet. It is the verified
environment and architecture foundation that future feature versions should build on.

## Dependency Notes

The `.AI_tool/frontend_README.md` originally described Vite 5 and jsPDF 2.x. This setup
uses patched versions where needed so the fresh baseline is audit-clean:

| Package | Version line | Reason |
|---|---|---|
| `vite` | `6.4.2` | Avoids known dev-server advisory in older Vite/esbuild stack |
| `vitest` | `3.2.4` | Compatible with Vite 6 |
| `jspdf` | `4.2.1` | Avoids vulnerable transitive DOMPurify versions |
| `jspdf-autotable` | `5.0.7` | Compatible with jsPDF 4 |

## Restore This Version

To inspect the exact tagged version:

```bash
git fetch --tags
git switch --detach v1.0.0
npm ci
npm run build
```

To create a working branch from this version:

```bash
git switch -c work/from-v1.0.0 v1.0.0
npm ci
```

To remove ignored generated files while restoring this version:

```bash
npm run version:use -- -Version v1.0.0 -CleanIgnored -Install
```

## Known Limitations

| Limitation | Impact |
|---|---|
| Feature implementation is not complete | Upload, analysis run, dashboard, charts, and PDF flows still need to be built |
| Backend is assumed, not bundled | API calls expect a FastAPI backend at `http://localhost:8000` through the Vite proxy |
| Starter test coverage is minimal | The testing environment is ready, but product tests need to grow with implementation |
| Versioning docs were added after the first tag | `v1.0.0` points to the exact baseline commit; the documentation lives on later `main` history |
